Road Traffic Accidents Detection Based On Crash Estimation

Introduction

This project aims to detect road traffic accidents based on crash estimation using machine learning models and data analysis techniques.

Installation

  1. Clone the repository: 
    git clone https://github.com/khaledsabry97/Road-Traffic-Accidents-Detection-Based-On-Crash-Estimation.git
  2. Change to the project directory: 
    cd Road-Traffic-Accidents-Detection-Based-On-Crash-Estimation
  3. Install the required dependencies: 
    pip install -r requirements.txt

Usage

To run the project, use the following command: 

python main.py
